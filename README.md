Dairy
=====

Simple Dairy Record Management System built with vanilla php as a Pro-Bono Project for a college student

## Features 
* Manage farmers
* Manage employees and their roles
* Record delivery
* Set payment rates
* Pay farmers
* View reports

### 1.  Manager
Has access to all the features
